# Plinko

## Setting it up Locally

### Backend

1. cd in folder

```bash
cd backend
```

2. Install dependencies

```bash
npm i
```

3. Build the application

```bash
npm run build
```

4. Start the application

```bash
npm run start
```

### Frontend

1. cd in folder

```bash
cd frontend
```

2. Install dependencies

```bash
npm i
```

3. Start the application

```bash
npm run dev
```
